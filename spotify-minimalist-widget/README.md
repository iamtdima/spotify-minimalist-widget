# Spotify Minimalist Widget for Übersicht

A clean, minimalist Spotify desktop widget for macOS using [Übersicht](https://icht.ai). 
Designed to match light/aesthetic desktop setups with a soft pink accent.

![Screenshot](Screenshot.jpeg)

## Features
- **Spotify Integration**: Displays current track and artist.
- **Instant Controls**: Play/Pause, Next, and Previous track (with zero-lag UI updates).
- **Aesthetic Design**: Border-only style with pink accents (`rgb(235, 155, 175)`), blur background, and vector icons.
- **Interactive**: Clicking the track info brings Spotify to the front.

## Installation
1. Install [Übersicht](https://icht.ai).
2. Download `index.jsx`.
3. Move it to your Übersicht widgets folder (usually `~/Library/Application Support/Übersicht/widgets/`).
4. Make sure Spotify is running!
